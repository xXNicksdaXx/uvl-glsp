# UVL Grammar and Metamodel

This module is derived from two upstream projects:

- https://github.com/Universal-Variability-Language/uvl-parser
- https://github.com/Universal-Variability-Language/java-fm-metamodel

It extends the original UVL metamodel/parsing implementation for the Behavioral Programming (BP) use case in this repository.
